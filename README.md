# Jarmo's Syndrome

Olet virus: tehtäväsi on levitä mahdollisimman paljon. Vastassasi on Jarmon immuunipuolustus. 
Puolustus vaikenee taso tasolta, mutta myös itse virus kasvaa ja muuttuu.
